# Inventario
1. Inventariar insumos
   - Nombre
   - Marca
   - Código serie
   - Comentario
2. Crear códigos
3. Imprimir y pegar códigos a los insumos

# Sistema
 - Ingresar Insumo
 - Buscar por insumo
 - Bitacora insumo
 - Agregar eventos a la bitacora (nombre, fecha, lugar, comentario)
   - Ingreso
   - En reparacion
   - Prestamo
   - Dar de baja
 - Filtrar por eventos
 - Reporte
